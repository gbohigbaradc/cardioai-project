## About

CardioAI is an Explainable Artificial Intelligence system developed as part of a dual-capstone research project. It applies machine learning, natural language processing, and explainable AI to two critical problems in Nigerian preventive healthcare:

1. **Cardiovascular Disease Risk Prediction** — identifying high-risk patients before symptoms become severe, enabling earlier clinical intervention.
2. **Patient Retention Prediction** — identifying which patients are likely to drop out of rehabilitation programs after their first visit, so clinicians can intervene proactively.

**Focus Area:** Nigeria — Lagos and Port Harcourt

## Features

| Feature | Description |
|---------|-------------|
| Cardiovascular Risk Prediction | Predicts heart disease risk using 4 ML models with SHAP explanations |
| Lifestyle Risk Index | Original composite behavioural risk score combining 5 clinical markers |
| Patient Retention Prediction | Predicts dropout risk from 10 operational and behavioural factors |
| Explainable AI (SHAP) | Feature attribution for every individual patient prediction |
| Clinical NLP + OCR | Extracts structured data from typed notes, scanned images, and PDFs |
| Tesseract OCR v5.5 | Reads printed and handwritten clinical documents |
| Interactive Dashboard | Streamlit-based web app for real-time clinical decision support |
| LLM Interpretation Layer | Natural language risk report generation (requires Anthropic API key) |

## Getting Started

### Prerequisites

- Python 3.9 or higher
- Anaconda (recommended) or pip
- Tesseract OCR (for local image OCR)

### 1. Clone the repository

```bash
git clone https://github.com/gbohigbaradc/cardioai-project.git
cd cardioai-project
```

### 2. Install Python dependencies

```bash
pip install -r requirements.txt
```

### 3. Install optional packages for full functionality

```bash
pip install xgboost imbalanced-learn spacy
python -m spacy download en_core_web_sm
```

### 4. Install Tesseract OCR (Windows)

Download and run the installer:
```
tesseract-ocr-w64-setup-5_5_0_20241111.exe
```
Available at: https://github.com/UB-Mannheim/tesseract/wiki

### 5. Place your dataset

Copy `heart.csv` into the `data/` folder:
```
data/heart.csv
```

### 6. Run the pipeline in order

```bash
python 01_preprocessing.py    # Clean data and create features
python 02_eda.py              # Generate analysis charts
python 03_model_training.py   # Train all models (5–15 minutes)
python 04_explainability.py   # Generate SHAP charts
python 05_nlp_ocr.py          # Test NLP pipeline
python 06_llm_interpretation.py  # Generate sample risk report
```

### 7. Launch the dashboard

```bash
streamlit run app/streamlit_app.py
```

Open your browser at: `http://localhost:8501`

## Data Sources

- **UCI Heart Disease Dataset** — Cleveland + Hungarian subsets combined (n=1,025). Source: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/45/heart+disease)
- **Synthetic Patient Retention Dataset** — 800 simulated patient records generated based on documented healthcare operations research factors.

## Technology Stack

**Machine Learning**
- scikit-learn — preprocessing, model training, evaluation
- XGBoost — gradient boosting classifier
- SHAP — explainable AI / feature attribution
- imbalanced-learn — SMOTE class balancing

**NLP & OCR**
- spaCy — named entity recognition
- Tesseract OCR v5.5 — image text extraction
- pdfplumber — PDF text extraction
- Regex NLP pipeline — clinical entity extraction

**Deployment**
- Streamlit — web dashboard framework
- Streamlit Community Cloud — free hosting
- GitHub — version control
- Anthropic API — LLM explanation layer (optional)
